**Subject:** Open-source Claude Code plugin built for PMs — PM AI Partner

Hey George,

I built an open-source Claude Code plugin specifically for Product Managers and wanted to see if it's a fit for the plugins list on prodmgmt.world/claude-code.

**PM AI Partner** — 10 agent skills, 4 workflow commands, and 3 automation hooks. One-command install:

```
npx pm-ai-partner@latest
```

What makes it different from other plugins:

- **PM-specific skills** — thought partner, technical analyst, writer, devil's advocate, builder, data analyst, product brief, meeting prep, stakeholder update, strategic clarity
- **Workflow commands** — `/pm:plan-week`, `/pm:write-brief`, `/pm:prep-meeting`, `/pm:audit-codebase`
- **Automation hooks** — quality gate for polished docs, draft status tracking, commit reminders
- **Multi-platform** — Claude Code, Cursor, and Codex
- **Full methodology** — backed by a complete PM-AI framework with a worked case study (OpenClaw deep dive)
- **MIT licensed** — fully open source

It's built on the idea that PM work is reconciling four sources of truth (code, docs, data, judgment) and each skill maps to one of these.

GitHub: https://github.com/ahmedkhaledmohamed/PM-AI-Partner-Framework
Plugin README: https://github.com/ahmedkhaledmohamed/PM-AI-Partner-Framework/tree/main/plugin

Would love your thoughts, and happy to adjust anything to fit the format of your curated list.

— Ahmed Khaled Mohamed
