## Hacker News

**Title:** Show HN: PM AI Partner – Claude Code skills for Product Managers

> I'm a PM at Spotify and built an open-source Claude Code plugin for product managers. One-command install: npx pm-ai-partner@latest
>
> The problem: AI coding assistants are built for engineers. PMs get generic chatbot responses instead of tools that match how we actually work.
>
> PM AI Partner gives Claude 12 agent skills (thought partner, technical analyst, devil's advocate, writer, builder, data analyst, product brief, meeting prep, stakeholder update, strategic clarity), 6 workflow commands, and 3 automation hooks.
>
> The design philosophy: PM work is reconciling four sources of truth — code (what CAN happen), docs (what's EXPECTED), data (what IS happening), and judgment (what SHOULD happen). Each skill maps to one of these.
>
> Example: /pm:devil-advocate doesn't just "challenge" an idea. It structures the challenge into what's strong, what's weak, likely objections by stakeholder type, and specific mitigations.
>
> Works with Claude Code, Cursor, and Codex. MIT licensed.
>
> GitHub: https://github.com/ahmedkhaledmohamed/PM-AI-Partner-Framework
