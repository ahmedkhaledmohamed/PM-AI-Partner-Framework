## Thread (7 tweets)

### 1/7 — Hook

AI coding assistants are built for engineers. Product Managers get generic chatbot responses.

I built an open-source Claude Code plugin that gives PMs 12 structured thinking modes instead.

One command: npx pm-ai-partner@latest

🧵

### 2/7 — The Problem

PM work is reconciling four sources of truth:
- Code → what CAN happen
- Docs → what's EXPECTED
- Data → what IS happening
- Judgment → what SHOULD happen

AI tools only help with one at a time. PMs need something that works across all four.

### 3/7 — What It Does

PM AI Partner gives Claude 12 "hats" it can wear:

🤔 Thought Partner — brainstorming
🔍 Technical Analyst — codebase exploration
😈 Devil's Advocate — stress-testing
✍️ Writer — strategy docs
🔨 Builder — prototypes
📊 Data Analyst — metrics
📋 Product Brief — PRDs
🎯 Meeting Prep — talking points
📢 Stakeholder Update — exec summaries
🧭 Strategic Clarity — team identity
🧪 Hypothesis Tester — A/B tests
🏁 Competitor Analyst — landscape mapping

### 4/7 — Not Just Prompts

Each skill has 5+ "Advanced Patterns" — encoded PM judgment, not templates.

Example: The Devil's Advocate skill knows to check for "the data gap" — proposals that sound data-driven but rest on data that doesn't exist yet.

That's the kind of thing a Senior PM catches. Now your AI does too.

### 5/7 — Workflow Commands

Plus 6 workflow commands:
- /pm:plan-week — weekly priorities with context
- /pm:write-brief — guided PRD creation
- /pm:prep-meeting — talking points + Q&A
- /pm:audit-codebase — PM-friendly code exploration
- /pm:competitor-scan — competitive landscape
- /pm:metrics-review — structured metrics analysis

### 6/7 — Install

Works with Claude Code, Cursor, and Codex.

```
npx pm-ai-partner@latest
```

30 seconds to install. MIT licensed. Free forever.
