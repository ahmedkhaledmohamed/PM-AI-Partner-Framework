## Plugin Name

PM AI Partner

## Short Description (1 line)

12 PM-specific agent skills, 6 workflow commands, and 3 automation hooks for Product Managers.

## Description

PM AI Partner is a Claude Code plugin that turns Claude into a structured thinking partner for Product Managers. Instead of generic responses, PMs get 12 distinct agent skills — each one a complete behavioral mode with encoded patterns from real PM practice.

**Skills (12):**
- Thought Partner — brainstorming and idea exploration
- Technical Analyst — codebase-to-product-insight translation
- Devil's Advocate — structured proposal stress-testing
- Writer — strategy docs, narratives, executive communication
- Builder — tools, prototypes, scripts, dashboards
- Data Analyst — metrics analysis and pattern finding
- Product Brief — guided PRD and feature spec creation
- Meeting Prep — talking points, Q&A preparation, role-play
- Stakeholder Update — status reports and executive summaries
- Strategic Clarity — team identity and boundary definition
- Hypothesis Tester — A/B test design and experiment analysis
- Competitor Analyst — competitive landscape mapping

**Workflow Commands (6):**
- `/pm:plan-week` — weekly planning with context awareness
- `/pm:write-brief [feature]` — guided product brief creation
- `/pm:prep-meeting [topic]` — meeting preparation with Q&A
- `/pm:audit-codebase [system]` — PM-friendly codebase exploration
- `/pm:competitor-scan [competitor]` — competitive analysis
- `/pm:metrics-review` — structured metrics review

**Automation Hooks (3):**
- Quality gate for polished documents
- Draft status tracking
- Commit reminders

**Also includes:** 17 MCP server configurations (GitHub, Linear, Slack, BigQuery, Google Drive, etc.)
