# Quick Setup Guide - Gemini Gem

## 5-Minute Setup

### Step 1: Open Gemini
Go to [gemini.google.com](https://gemini.google.com)

### Step 2: Create New Gem
- Click **"Gem Manager"** (sidebar or menu)
- Click **"Create new Gem"**

### Step 3: Name Your Gem
```
Name: PM AI Partner
Description: Thinking partner for PMs using the PM AI Partner Framework
```

### Step 4: Add Instructions
Copy everything from `instructions.md` and paste into the instructions field.

### Step 5: Save & Test
Click **"Save"** then test:
```
"What are the six agent modes?"
```

Expected: Lists all six modes with descriptions.

### Step 6: Start Using
Invoke modes by name:
```
"Act as Technical Analyst. Map API endpoints in this code."
"Act as Writer. Help me draft a team charter."
```

## Troubleshooting

**Gem not following instructions?**
- Invoke mode explicitly: "Act as [Mode Name]"
- Provide more context in your prompt

**Generic responses?**
- Be specific about what you need
- Reference the framework: "Using Strategic Clarity workflow..."

**Can't find Gem Manager?**
- Check if Gems are available in your region
- Try accessing from desktop (not mobile)
- Enable in Google Labs settings
